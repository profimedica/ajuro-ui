# ajuro-ui
Ajuro UI Elements

=========

A small library that adds commas to numbers

## Installation

  `npm install @profimedica/ajuro-ui`

## Usage

    var numFormatter = require('@profimedica/ajuro-ui');

    var formattedNum = numFormatter(35666);
  
  
  Output should be `35,666`


## Tests

  `npm test`
